# AI-Powered Customer Care Work Buddy

An intelligent customer care assistant integrating text, voice, and emotion recognition.

## Features
- AI chatbot for customer support  
- Sentiment and emotion analysis using TensorFlow  
- Video and audio input support  
- JWT-based authentication  
- Built with Django (backend) and Next.js (frontend)

## Setup Instructions
1. Clone this repository  
2. Install dependencies for both backend and frontend.
3. Run both servers (`python manage.py runserver` / `pnpm dev`)
4. The docker-compose.yaml file can be used to run the necessary tools for the DB.
